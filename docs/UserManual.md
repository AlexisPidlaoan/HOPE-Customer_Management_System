# User Manual

## 👋 Overview
This system is a Customer Management Platform used to manage business operations such as customers, services, and appointments.

---

## 🔐 Login
- Users must log in before accessing the dashboard
- Authentication is required for security

---

## 🏠 Dashboard
- Displays summary of system data
- Navigation is available through the sidebar

---

## 👤 Customers
- Add, view, and manage customer information

---

## 🛠 Services
- Manage available services offered by the system

---

## 📅 Appointments
- Schedule and track appointments

---

## 🚪 Logout
- Users can securely log out from the system
